## Introduction


This is 4chan Vocaroo URL replacer, a userscript for Firefox, Chrome and Opera... Just any browser that has a userscipt extention

It converts text or a link such as http://vocaroo.com/?media=vT1plxX90UX55rEGT, vocaroo.com/17luWJaGOLFi and https://voca.ro/17luWJaGOLFi

into an embedded object for easy playback.


You can now also click a button too load the embed rather than all the embeds just apearing :3
